Uma página simples hospedada com GitHub Pages

Bem-vindo!

Esta é uma página estática de exemplo criada para validar a integração com o GitHub Pages. Ela usa apenas HTML e CSS e pode ser publicada diretamente a partir deste repositório.

Como publicar

  1. Abra as configurações do repositório no GitHub e acesse a seção Pages.
  2. Em Source, selecione o branch desejado (por exemplo, main) e a pasta / (raiz) como origem.
  3. Salve as alterações e aguarde alguns minutos para a página ser disponibilizada.

Próximos passos

Personalize este layout para refletir o seu projeto. Você pode adicionar novas seções, incluir imagens ou mesmo criar páginas adicionais na pasta docs/ caso prefira manter a raiz do repositório apenas para código-fonte.
